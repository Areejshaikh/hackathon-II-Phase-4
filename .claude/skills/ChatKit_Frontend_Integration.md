# ChatKit Frontend Integration

## Purpose
OpenAI ChatKit UI ko frontend mein integrate karna taake user natural language mein chatbot se baat kar sake.

## Scope
- Next.js pages/components
- ChatKit setup + domain key
- Existing dashboard/login flow extension

## Responsibilities / Key Capabilities
1. ChatKit component setup karna (NEXT_PUBLIC_OPENAI_DOMAIN_KEY use)
2. Domain allowlist OpenAI settings mein add karna
3. Chat interface banana (message input + response display)
4. /api/{user_id}/chat endpoint se messages bhejna
5. Tool results render karna (task list, confirmations)
6. User name/email chat header mein show karna
7. Responsive + sky-blue theme maintain karna
